Tick items in or out of stock, edit prices and quantities, then save. Saved changes update the public stock list on this device immediately — use "Download data.json" to publish the changes for everyone (see instructions below).

How publishing works: this page saves changes to this device only. To make new stock visible to everyone on the live website, click "Download data.json", then replace the data.json file in the GitHub repo with the downloaded one (upload it via github.com → your repo → "Add file" → "Upload files"). The site updates automatically a minute or two after that.
